String
    字符串
构造函数
    new String(s) //构造函数
    function string(s)  //转换函数
返回
    当使用new操作符将String作为一个构造函数使用时，将会返回一个String对象，内容为字符串s或s的字符串表示。不带new操作符时，只是简单的将s转换为原始字符串并返回转换后的值。
属性
    length
        该字符串中的字符数，即字符串的长度
方法
    charAt()
        返回一个字符串中指定位置的字符
    charCodeAt()
        返回一个字符串中指定位置的字符的编码
    concat()
        将一个或多个值连接城一个字符串
    indexOf()
        在指定字符串中寻找一个字符或子串
    lastIndexOf()
        在指定字符串中从向右开始寻找一个字符或子串
    localeCompare()
        使用本地定义的顺序比较字符串
    match()
        使用正则表达式执行模式匹配
    replace()
        使用正则表达式查询或替换操作
    search()
        在一个字符串中查找匹配某个正则表达式的子串
    slice()
        返回一个字符串的一个切片或子串
    split()
        将字符串分割为由字符串组成的数组
    substr()
        提取字符串的一个子串，substring的一个变体
    substring()
        提取字符串的一个子串
    toLowerCase()
        返回字符串的一个副本，其中全部转化为小写
    toUpperCase()
        返回字符串的一个浮肿，其中字符全部转换为大写
    toString()
        返回原始的字符串值
    trim()
        返回字符串的副本，其中前后面的空白字符将删除
    valueOf()
        返回原始的字符串值
静态方法
    String.fromCharCode()
        使用作为参数传入的字符编码创建一个新的字符串
